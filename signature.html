How to add this to Zoho Mail

  1. Open this file in your browser (double-click signature.html)
  2. In Zoho Mail, go to Settings → Signatures → Add New Signature
  3. Click the </> Source or HTML button in the signature editor
  4. Delete any placeholder content in the HTML source box
  5. Copy only the <table>...</table> block from this file's source and paste it in
  6. Click Save, then set it as your default signature

If Zoho doesn't show an HTML source button, switch the editor to Rich Text mode, select all content in the preview above, copy it (Ctrl+C), and paste directly into the signature box.
